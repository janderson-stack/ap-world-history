Enter the Teacher Hub

This page is a static prototype gate. It keeps students from casually entering the Teacher Hub, but it is not a secure authentication system for private student data.

Security note: true student-response protection should happen through a protected Google Apps Script, authenticated Google account access, or another secure backend layer. Do not store private student data directly in GitHub Pages files.

Rollback-safe build approach

The Teacher Hub is isolated inside /teacher/ and separate assets/css, assets/js, and assets/data/teacher files. This keeps the student-facing BeHistorical lesson renderer untouched during prototype development.
